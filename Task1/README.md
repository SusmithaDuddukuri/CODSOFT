# Number Guessing Game 🎮

## CodSoft Java Programming Internship - Task 1

### 📌 Description

This project is a console-based **Number Guessing Game** developed using Java as part of the **CodSoft Java Programming Internship**.

The program generates a random number between **1 and 100**, and the user must guess the number within a limited number of attempts. After each guess, the program provides feedback indicating whether the guess is too high or too low. The game also supports multiple rounds and score tracking.

---

### ✨ Features

* Random number generation
* User input handling
* High/Low feedback system
* Limited attempts per round
* Multiple game rounds
* Score calculation
* Interactive console interface
* User-friendly gameplay
