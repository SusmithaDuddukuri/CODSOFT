# ATM Interface 🏧

## CodSoft Java Programming Internship - Task 3

### 📌 Description

This project is a console-based **ATM Interface** developed using Java as part of the **CodSoft Java Programming Internship**.

The program simulates basic ATM operations by allowing users to check their account balance, deposit money, and withdraw money through a menu-driven interface. The system is designed using Object-Oriented Programming concepts with separate classes for the ATM machine and the user's bank account. It also includes input validation and displays appropriate messages based on transaction success or failure.

---

### ✨ Features

* Menu-driven ATM interface
* Check account balance
* Deposit money functionality
* Withdraw money functionality
* Input validation for transactions
* Insufficient balance handling
* Object-Oriented design (ATM & BankAccount classes)
* User-friendly console output
