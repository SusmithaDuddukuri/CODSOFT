# Currency Converter 💱

## CodSoft Java Programming Internship - Task 4

### 📌 Description

This project is a console-based **Currency Converter** developed using Java as part of the **CodSoft Java Programming Internship**.

The program allows users to convert an amount from one currency to another using predefined exchange rates. Users can select a base currency and a target currency from supported options such as **USD, INR, and EUR**. The application performs the conversion instantly and provides an option for multiple conversions in a single session.

---

### ✨ Features

* Currency conversion between USD, INR, and EUR
* Predefined exchange rates
* User-friendly console interface
* Multiple currency combinations supported
* Continuous conversion option
* Input handling using Scanner
* Formatted output display
* Simple and efficient implementation
