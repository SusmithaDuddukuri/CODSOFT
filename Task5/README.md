# Student Management System 🎓

## CodSoft Java Programming Internship - Task 5

### 📌 Description

This project is a console-based **Student Management System** developed using Java as part of the **CodSoft Java Programming Internship**.

The program allows users to manage student records through a menu-driven interface. Users can add new students, view all stored student details, and remove students using their roll numbers. The project is implemented using Object-Oriented Programming (OOP) concepts with separate classes for student data and management operations.

---

### ✨ Features

* Add new student records
* View all student details
* Remove students by roll number
* Menu-driven console interface
* Object-Oriented Programming implementation
* Dynamic student storage using ArrayList
* User-friendly formatted output
* Simple and efficient record management
