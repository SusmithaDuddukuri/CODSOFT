# Student Grade Calculator 🎓

## CodSoft Java Programming Internship - Task 2

### 📌 Description

This project is a console-based **Student Grade Calculator** developed using Java as part of the **CodSoft Java Programming Internship**.

The program allows users to enter marks obtained in multiple subjects (out of 100). It calculates the **total marks**, **average percentage**, assigns a **grade**, and displays a **performance remark** based on the calculated average. The application also includes input validation and a retry option for multiple calculations.

---

### ✨ Features

* Multiple subject marks input
* Input validation (0–100 range)
* Automatic total marks calculation
* Average percentage computation
* Grade assignment system
* Performance remarks display
* Retry option for repeated calculations
* Clean formatted output
